# Going live

In the previous topics, we were able to create our first payment, retrieve all of a entity's debts
and refresh a liability using Method's Payments, Data and Sync products.

Because this guide was completed in the `development` environment all the requests and data were faked for simplicity.s

Before moving to `production` we recommend performing full end-to-end test in the `sandbox` environment.
