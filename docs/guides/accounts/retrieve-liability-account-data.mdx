# Retrieve liability account data

Accounts connected through [Method Data](/guides/quick-start/retrieve-an-entitys-debts)
automatically have enhanced and comprehensive liability data, and are ready to receive payments.

:::caution
Only accounts with the `data:retrieve` capability activated can continue
with this guide. See the [Method Data](/guides/quick-start/retrieve-an-entitys-debts) flow
section if you wish to follow this guide.
:::

---

### Retrieve liability account
This request retrieves a liability account created from the Method Data flow.

➡️ See [liability account types](/api/core/accounts/object#liability-account-types) to learn more.
