# Liability account data

Accounts connected through [Method Data](/guides/quick-start/retrieve-an-entitys-debts)
automatically have basic liability data, and are ready to receive payments.

In order to retrieve enhanced / expanded liability data you must initiate a sync using [Method Sync](/guides/quick-start/sync-a-liability)

---

### Sync a liability's data

‚ú® Method Sync allows you to retrieve real-time enhanced data for any liability created in Method. Method Sync is available
for a fast growing list of `merchants`.

With Sync, you can fetch key liability details such as `interest_rate_percentage`, `payoff_amount`, `balance` and much more
in real-time directly from the servicer or financial institution. ‚û°Ô∏è See all available data fields below per liability type.

üí° To check the status of a sync use the data_status and data_last_successful_sync in the account.liability response.
