# Exchange Public Account Token

**About**: Exchanges the `public_account_token` for an `account`. This is the final step when linking an account using Method Link.

**API Method**: `POST /link/token`

### Request

#### Example
```shell
curl https://production.methodfi.com/link/exchange \
  -X POST \
  -H "Authorization: Bearer sk_WyZEWVfTcH7GqmPzUPk65Vjc" \
  -H "Content-Type: application/json" \
  -d '{
    "public_account_token": "pk_acc_iZKeBTmnmw3aSGePcfdweUEJTBXwsfR4"
  }'
```
