# Update an individual auth session

**About**: Update an auth session for an entity to automatically retrieve their
debt accounts.

**API Method**: `PUT /entities/:entityId/auth_session`

:::caution
This endpoint is only accessible to entities that have the `data:retrieve` capability
in `available_capabilities`.
:::
