# Create an individual auth session

**About**: Creates a new auth session for an entity to automatically retrieve their
debt accounts.

:::caution
This endpoint is only accessible to entities that have the `data:retrieve` capability
in `available_capabilities`.
:::

:::info
If your organization is enrolled in Method's non-KBA authentication, you can skip the auth session and
[retrieve the accounts for an entity](/api/core/accounts/list).
:::
