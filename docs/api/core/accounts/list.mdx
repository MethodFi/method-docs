# List all accounts

**About**: Returns all the accounts associated with your team, or an empty array if none have been created.

**API Method**: `GET /accounts`

### Request

#### Example
<AccountsList />

#### Parameters

| Name        | Required | Description                                                     |
|-------------|----------|-----------------------------------------------------------------|
| `holder_id` |          | The id of the `Entity` whose accounts to retrieve.              |
| `type`      |          | The type of accounts to retrieve. Options: `ach` or `liability` |
