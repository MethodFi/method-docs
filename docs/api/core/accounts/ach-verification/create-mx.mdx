# Create an MX verification

**About**: Verify an ACH account by providing the raw account and transaction data from MX.

**API Method**: `POST /accounts/:account_id/verification`

### Request

#### Example
<VerificationCreateMX />

#### Parameters

| Name              | Type       | Required | Description                                              |
|-------------------|------------|----------|----------------------------------------------------------|
| `type`            | `string`   | ✅        | The type of verification. Option for MX: `mx`.           |
| `mx`              | `object`   | ✅        | Object containing account data from MX.                  |
| `mx.account`     | `object`   | ✅        | The raw account data received from MX. See https://docs.mx.com/atrium#endpoints_accounts_read_account                   |
| `mx.transactions` | `object[]` | ✅        | The a list of the raw transaction data received from MX. See https://docs.mx.com/atrium#endpoints_accounts_list_account_transactions |
