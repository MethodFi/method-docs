# Create a Plaid verification

**About**: Verify an ACH account by providing the raw balance and transaction data from Plaid.

**API Method**: `POST /accounts/:account_id/verification`

### Request

#### Example
<VerificationCreatePlaid />

#### Parameters

| Name                 | Type       | Required | Description                                                                                                   |
|----------------------|------------|----------|---------------------------------------------------------------------------------------------------------------|
| `type`               | `string`   | ✅        | The type of verification. Option for Plaid: `plaid`.                                                          |
| `plaid`              | `object`   | ✅        | Object containing account data from Plaid.                                                                    |
| `plaid.balances`     | `object`   | ✅        | The raw balance data received from Plaid. See https://plaid.com/balance/#balance-data                    |
| `plaid.transactions` | `object[]` | ✅        | The a list of the raw transaction data received from Plaid. See https://plaid.com/transactions/#overview |
