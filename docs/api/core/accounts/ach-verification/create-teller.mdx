# Create a Teller verification

**About**: Verify an ACH account by providing the raw balance and transaction data from Teller.

**API Method**: `POST /accounts/:account_id/verification`

### Request

#### Example
<VerificationCreateTeller />

#### Parameters

| Name                 | Type       | Required | Description                                                                                                   |
|----------------------|------------|----------|---------------------------------------------------------------------------------------------------------------|
| `type`               | `string`   | ✅        | The type of verification. Option for Teller: `teller`.                                                          |
| `teller`              | `object`  | ✅        | Object containing account data from Teller.                                                                    |
| `teller.balances`     | `object`  | ✅        | The raw balance data received from Teller. See https://teller.io/docs/api/2020-10-12#account_balances_anchor                    |
| `teller.transactions` | `object[]`| ✅        | The a list of the raw transaction data received from Teller. See https://teller.io/docs/api/2020-10-12#transactions_anchor |
