# List all merchants

**About**: Returns the most popular `liability` merchants that are payable using Method's API.

**API Method**: `GET /merchants`

:::info Important
Only the most popular merchants are returned at once in the call to `GET /merchants`. Use the available filters/query parameters to narrow your search and find many more merchants.
:::

### Request

#### Example
<MerchantsList />

#### Parameters

| Name                   | Type     | Required | Description                                                                                                       |
|------------------------|----------|----------|-------------------------------------------------------------------------------------------------------------------|
| `name`                 | `string` |          | Name of the merchant to find.                                                                                     |
| `creditor_name`        | `string` |          | The name of the creditor as seen in a credit report. Ex. `JPMCB CARD` pertains to a Chase Credit Card. |
| `provider_id.plaid`    | `string` |          | Plaid id for the merchant to find.                                                                                |
| `provider_id.mx`       | `string` |          | MX id for the merchant to find.                                                                                   |
| `provider_id.finicity` | `string` |          | Finicity id for the merchant to find.                                                                             |
