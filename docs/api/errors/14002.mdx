# 14002 - Unauthorized

```json
{
  "type": "PAYMENT_REVERSAL_FAILED",
  "code": 14002,
  "sub_type": "PAYMENT_REVERSAL_UNAUTHORIZED",
  "message": "Payment reversal failed due to a report by the account holder as unauthorized."
```

### Causes
- The account holder reported to their account's
financial institution that a transaction processed through Method was unauthorized.

### Resolution
- The failed reversal will be manually resolved by the Method team.
