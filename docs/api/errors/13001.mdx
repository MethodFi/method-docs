# 13001 - Attempt Limit Reached

```json
{
  "type": "ACCOUNT_VERIFICATION_DISABLED",
  "code": 13001,
  "sub_type": "ACCOUNT_VERIFICATION_ATTEMPT_LIMIT_REACHED",
  "message": "Account verification was disabled due to the maximum verification limit (5) being reached."
}
```

### Causes
- Five unsuccessful attempts have been made to verify the micro-deposit amounts.

### Resolution
- Both the account verification and the account will be disabled.
