# The BIN object

A BIN (Bank Identification Number) is the initial set of four to six numbers that appear on a payment card. This provides information regarding the issuer of the card.

### BIN object

#### Example

```json
{
  "id": "bin_fWXVJF8HWYSdQ",
  "bin": "414720",
  "brand": "visa",
  "issuer": "CHASE BANK USA, N.A.",
  "type": "credit",
  "category": "SIGNATURE",
  "bank_url": "www.chase.com",
  "sample_pan": "4147206867378753"
}
```

#### Attributes

| Name       | Type                 | Description                                                                                           |
|------------|----------------------|-------------------------------------------------------------------------------------------------------|
| id         | `string`, `nullable` | A unique identifier for a `BIN`.                                                                      |
| bin        | `string`, `nullable` | The 6-digit number.                                                                                   |
| brand      | `string`             | The payment card brand. One of the following: `amex`, `diners_club`, `discover`, `mastercard`, `visa` |
| issuer     | `string`, `nullable` | The issuer of the payment card.                                                                       |
| type       | `string`             | The type of payment card. Either `credit` or `debit`                                                  |
| category   | `string`, `nullable` | The payment card category.                                                                            |
| bank_url   | `string`, `nullable` | The website URL of the associated bank.                                                                |
| sample_pan | `string`, `nullable` | An example Primary Account Number for the bin.                                                         |
